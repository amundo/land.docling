GitHub
An internet host (github.com) which provides storage and infrastructure for managing software development projects using the Git version-control management system.

host
A computer which is connected to the Internet. Usually, hosts can be reached via a URL.

URL
The address of a given file or other resource on a particular host on the internet.

domain name
A textual “name” or “address” for an web or internet host. For example, <www.example.com> or <example.com>.

browser
Also, web browser. A program which is responsible for rendering Web content in a usable form. Typically a browser retrieves content from a remote web server.

CSS
Cascading Style Sheets — the web platform’s language for defining presentation of HTML documents.

HTML
Hypertext Markup Language — a markup language used to create documents which can be presented by a web browser.
